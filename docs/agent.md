# The Agent Class

```yaml
multimodal_agent.agent_core.MultiModalAgent
```

## Commands

### Ask a text question

```bash
agent ask "What is quantum entanglement?"
```

### Describe an image

```bash
agent image photo.jpg "what is in this photo?"
```

### Start chat session

```bash
agent chat
```

### Show version

```bash
agent --version
```

### Enable debug logs

```bash
agent --debug ask "hello"
```
