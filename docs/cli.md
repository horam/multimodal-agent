# CLI Usage

The package installs a command-line tool called: agent

## Commands

### Ask a text question

```bash
agent ask "What is quantum entanglement?"
```

### Describe an image
```bash
agent image photo.jpg "what is in this photo?"
```
### Start chat session
```bash
agent chat
```
### Show version
```bash
agent --version
```
### Enable debug logs
```bash
agent --debug ask "hello"
```
