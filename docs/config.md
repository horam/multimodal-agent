# Config


| Field     | Description |
| ---------- | --------- |
| chat_model | Model used for text-only queries and chat   |
| image_model | Model used for image + text reasoning      | 
| embedding_model |Model used for RAG embeddings          |
	

	

Example block:
```yaml
chat_model: gemini-2.5-flash
image_model: gemini-2.0-flash
embedding_model: text-embedding-004
```
